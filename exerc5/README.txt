Nesse exercício foi pedido que fosse feita uma lista de pessoas,
cada pessoa é do tipo pessoa, determinada por uma struct.
Essa lista deve ser alocada dinamicamente e deve armazenar quantas
pessoas o usuário desejar.

O Dr. Memory parece estar apresentando erros de acesso não inicializado
independente do que eu faço no programa, mesmo se eu apenas entrar e sair do programa.

Este zip contém o arquivo .c com o código em si, este README, um arquivo executável
e uma pasta com os testes usando Dr. Memory.

Caso o executável apresente problemas usar "gcc exerc5.c -Wall -g -o exerc5.exe".

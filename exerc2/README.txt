Nesse exercício foi pedido que fosse feita uma lista de nomes,
que pudessem ser adicionados, removidos e mostrados à vontade do usuário.
Essa lista deve ser armazenada em uma string que aumenta e diminui de acordo com 
a quantidade de nomes adicionados ou removidos.

Infelizmente não fui capaz de implementar a deleção de nomes,
e o Dr. Memory apresentou erros que não consegui consertar.

Esta zip contém o arquivo .c com o código em si, este README, um arquivo executável
e uma pasta com os testes usando Dr. Memory e o vídeo.

Caso tal arquivo apresente problemas usar "gcc exerc2.c -Wall -g -o exerc2.exe".
